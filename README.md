# PlayFabBook
本リポジトリは PlayFab ゲーム開発テクニックのサンプルコードのリポジトリです。

本アプリケーションのサンプルコードはご自由にご利用ください。
ただし同梱されるアセット/ライブラリの扱いはそれぞれのライセンスを遵守してください。
